おすすめ

おすすめのツール、サービス、R パッケージ、Zotero プラグイン、本の一覧。

個人で使っている、または後で参照したいツール・サービス・本を用途別にまとめています。 オンラインで利用できるものを中心に、随時更新します。

ツール・サービス

ノート・文献管理

  • Obsidian: Markdown 形式でノートを取れるノートアプリ。リンク機能やタグ機能が充実している。
  • Zotero: 参考文献管理ツール。
  • Elicit: AI を活用したリサーチツール。文献検索や要約などに利用できる。
  • Google Scholar | Labs: Google Scholar の実験的な検索機能。
  • Undermind: 研究文献を探索するための AI 検索ツール。
  • ResearchRabbit: 文献同士の関係をグラフで可視化し、関連文献を探しやすくするツール。

開発・ドキュメント作成

  • GitHub: ソースコード管理とプロジェクト公開のためのサービス。
  • Quarto: ドキュメント作成ツール。R Markdown の後継として利用している。
  • Positron: データサイエンス向け IDE。最近は RStudio よりもこちらを使うことが多い。

情報収集・タスク管理

  • Raindrop.io: ブックマーク管理ツール。
  • Karakeep: セルフホストできるブックマーク管理ツール。Raindrop.io の代替候補として導入したが、まだ十分には活用できていない。
  • VisiOS: ブックマークやタブの管理ツール。
  • Inoreader: RSS リーダー。
  • Todoist: タスク管理ツール。

画像・動画・配信

  • Eagle: 画像管理ツール。画像をタグで管理できる。
  • Affinity: 画像や写真の編集ソフト。Adobe Photoshop や Illustrator の代替として利用できる。
  • Kdenlive: 動画編集ソフト。
  • OBS Studio: 画面録画・配信用ソフト。
  • Keyviz: オープンソースのキーボードビジュアライザー。

セキュリティ・ネットワーク

  • Bitwarden: パスワード管理ツール。
  • Proton Mail: セキュアなメールサービス。
  • Proton Authenticator: 2 段階認証アプリ。
  • Tailscale: VPN サービス。
  • NordVPN: VPN サービス。Proton VPN への乗り換えを検討中。
  • RustDesk: リモートデスクトップソフト。AnyDesk の代替。
  • Brave: プライバシー重視のウェブブラウザ。
  • Element: オープンソースのチャットアプリ。Matrix プロトコルを使用。

AI

  • LM Studio: ローカルで動作する AI チャットツール。ローカル LLM を試すときに便利。

R パッケージ

  • tmap: 地図の作成に特化した R パッケージ。地理空間データの可視化に便利。
  • rnaturalearth: 自然地理データを提供する R パッケージ。世界の国境や地形などを簡単に取得できる。
  • cols4all: カラーパレットを提供する R パッケージ。色覚多様性に配慮したカラーパレットも利用できる。

Zotero プラグイン

  • Better BibTeX for Zotero: Obsidian など外部ソフトウェアとの連携に便利。
  • Ethereal Style: Zotero の UI を拡張するプラグイン。
  • ZotCard: カード形式のメモやメモテンプレートを作成できるプラグイン。
  • Zotero Reading List: 未読・既読・途中など、文献の閲覧状況を管理するプラグイン。
  • ZotSeek: 類似文献の検索を補助するプラグイン。
  • Zotero Better Notes: Zotero のメモ機能を強化するプラグイン。
  • Zotodo: Todoist などのタスク管理アプリと Zotero を連携するプラグイン。
  • Zotero Reference: 引用文献へのアクセスや文献リスト作成を補助するプラグイン。

本・オンライン教材

植物・形態

  • Manual of Leaf Architecture: 葉の形態に関するマニュアル。葉の形態を理解するための基本的な知識がまとまっている。ライセンスは CC BY-NC 4.0。
  • Contributions to Morphometrics: 形態測定学に関する古典的な資料。

R

オンラインで読める R 関連の良質な本です。

  • R for Data Science (2e): R を使ったデータサイエンスの入門書。データの取り扱い、可視化、モデリングなどが網羅されている。ライセンスは CC BY-NC-ND 3.0。
  • R Graphics Cookbook, 2nd edition: R のグラフィックスに関するレシピ集。ggplot2 を使ったグラフの作成方法が豊富に紹介されている。
  • Cookbook for R: R の様々なタスクに関するレシピ集。ライセンスは CC0。
  • Tidy Modeling with R: tidymodels を使ったモデリングの入門書。
  • Efficient R programming: R を効率よく書くための実践的な本。
  • Big Book of R: R 関連のオンライン教材を集めたリンク集。
  • Mastering Shiny: Hadley Wickham 氏による Shiny の入門書。
  • Deep Learning with R: R で深層学習を扱うための本。
  • blogdown: Creating Websites with R Markdown: blogdown を使ったウェブサイト作成の本。
  • Mastering Spark with R: R から Spark を扱うための本。
  • R Packages (2e): R パッケージ開発の入門書。
  • ggplot2: Elegant Graphics for Data Analysis (3e): ggplot2 の入門書。

Python

  • Think Python: Python の入門書。
  • Think Python, 2nd edition: Think Python の第 2 版。
  • Cracking Codes with Python: 暗号を題材に Python を学ぶ本。
  • Beyond the Basic Stuff with Python: Python の基礎の次に読む本。
  • The Big Book of Small Python Projects: 小さな Python プロジェクトを通して学ぶ本。
  • Python for Data Analysis, 3E: pandas を中心としたデータ分析の定番書。
  • データ分析のための統計学入門: データ分析に必要な統計学の入門資料。
  • IPython Cookbook, Second Edition: IPython と Python の実践的なレシピ集。
  • Automate the Boring Stuff with Python: Python による自動化の入門書。

統計・可視化・コンピュータサイエンス

  • An Introduction to Statistical Learning: 統計学習の入門書。
  • Fundamentals of Data Visualization: データ可視化の基本原則を解説した本。ライセンスは CC BY-NC-ND 4.0。
  • Think Bayes 2: ベイズ統計の入門書。
  • Think Stats, 3rd edition: 統計の基礎を Python で学ぶ本。
  • Foundations of Computer Vision: コンピュータビジョンの基礎を解説した本。
  • Open Data Structures: データ構造の入門書。
