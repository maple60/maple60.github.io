## English

### About

This is a personal blog and notebook about programming, data analysis, reproducible documents, and research tooling. I mainly write notes about R, Python, Quarto, visualization, website publishing, literature management, and research workflows.

### Topics

- **R and data analysis**: Visualization, maps, image processing, statistics, and reproducible analysis workflows.
- **Quarto and GitHub Pages**: Website publishing, rendering settings, code links, GitHub Actions, and Japanese text support.
- **Python and tooling**: Environment management with `uv` and small automation notes.
- **Research workflow**: Zotero, literature notes, citation management, and data organization.

### Navigation
